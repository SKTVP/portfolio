About Me

I'm a full-stack developer who builds high-performance, production-ready web applications from the ground up. I specialise in Rust backends and React frontends, delivering software that is fast, secure, and maintainable.

Projects

UniLearnIt

Production SaaS

A full-stack private tutoring platform serving 500+ university students yearly in Greece — built solo from scratch.

What it does

  • Real-time chat between students and tutors via WebSocket actors with Redis pub/sub for horizontal scaling
  • Video streaming with DRM protection and per-user dynamic watermarks
  • Stripe payment processing with semester-based pricing, webhook verification, and admin-toggleable billing
  • Multi-role access control — Admin, Tutor, and Student dashboards with JWT-secured routes
  • Zoom integration — automated webhook handling for lecture recordings with HMAC validation
  • Course management — tables, PDF uploads with watermarking, file collections, rich text editing (Tiptap)
  • Student enrollment system — department-based course discovery, enrollment requests, device management
  • Admin panel — full CRUD for courses, users, pricing, universities, departments, and analytics
  • Email service — bulk course announcements via SMTP with HTML templates

Architecture highlights

  • Domain-driven modular backend with 15+ feature modules
  • Solved a CPU bottleneck — PDF-to-image conversion was blocking the server; redesigned the pipeline to upload PDFs asynchronously and convert in the background, cutting CPU cost significantly
  • CI/CD with GitHub Actions, Docker, and Google Cloud Run
  • Prometheus metrics and structured logging for production observability
